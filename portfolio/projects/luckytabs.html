LuckyTabs

Mobile sustainability application to keep track of eco-friendly habits.

Project Overview

About This Project

LuckyTabs is a sustainability app that helps users to manage and be motivated to keep up eco-friendly habits. The idea of this app is to reward users when they recycle their bottles or cans when using the Reverse Vending Machines.

Technologies Used

Flutter Dart Firebase Android Studio
